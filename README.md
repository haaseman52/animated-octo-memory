# Animated Octo Memory

## Project Overview
This project aims to create an engaging and interactive memory game featuring animated octopuses. Players will have to match pairs of octopus cards within a limited timeframe, testing their memory and speed.

### Features
- Engaging animations
- Multiple difficulty levels
- Interactive gameplay
- High score tracking

## Getting Started
To get started, clone the repository and open the `index.html` file in your browser.
